## 🚀 English Description
**VibePalette** is a smart, ultra-fast color utility designed for **Vibe Coders**. It provides instant, AI-ready color prompts for both Light and Dark modes with a single click. No more manual hex hunting—just copy the prompt and feed it to your favorite AI tool (Lovable, ChatGPT, Cursor, etc.).

### ✨ Key Features
- **45+ Professional Palettes:** Hand-picked tones for modern UIs.
- **AI-Ready Prompts:** Copies full context (Light/Dark/Gradients) instantly.
- **Fully Responsive:** Beautiful UI for both Mobile and Desktop.
- **Bilingual Support:** Switch between Arabic and English seamlessly.

## 🚀 الوصف بالعربية
**VibePalette** هي أداة ذكية وسريعة جداً مصممة لمطوري الـ **Vibe Coding**. توفر لك "برومبت" ألوان جاهز للذكاء الاصطناعي يدعم الوضعين (النهاري والليلي) بضغطة زر واحدة. وداعاً للبحث اليدوي عن أكواد الهكس—فقط انسخ الأمر وأعطه لأدواتك الذكية (Lovable, ChatGPT, Cursor).

### ✨ المميزات
- **أكثر من 45 لوحة لونية:** ألوان مختارة بعناية للواجهات العصرية.
- **أوامر جاهزة للذكاء الاصطناعي:** نسخ كامل السياق (نهاري/ليلي/تدرج) فوراً.
- **تجاوب كامل:** واجهة جميلة تعمل على الجوال والمكتبي.
- **دعم لغتين:** تبديل سلس بين العربية والإنجليزية.
